# ⚖️ AbogApp - Gestión Legal para Estudios Jurídicos

**AbogApp** es una aplicación web pensada para ayudar a abogados y estudios jurídicos a organizar y gestionar fácilmente sus clientes, causas (expedientes) y tareas, sin necesidad de conocimientos técnicos avanzados.

---

## 🚀 Funcionalidades principales

- 📋 Registro de clientes con ficha editable
- 📁 Gestión de expedientes asociados a cada cliente
- 📝 Registro de tareas asociadas a expedientes o tareas administrativas
- 📊 Dashboard con resumen visual (clientes, expedientes, tareas y urgencias)
- 🔄 Importación y exportación de datos en formato JSON
- 📆 Preparado para integrar Google Calendar
- 📱 Adaptado para uso en dispositivos móviles

---

## 🛠 Cómo usar

1. Clona el repositorio o descarga los archivos.
2. Abre `index.html` en tu navegador.
3. Usa los formularios para registrar clientes, expedientes y tareas.
4. Exporta tu base de datos con el botón en la pestaña de **Utilidades**.
5. Puedes importar una base existente en formato `.json`.

---

## 📦 Requisitos

Este proyecto **no necesita instalación**. Funciona 100% en el navegador.

---

## 🧪 Datos de prueba

Puedes cargar el archivo `abogapp-backup.json` desde la pestaña **🛠 Utilidades > Importar base de datos** para ver la app en funcionamiento con datos simulados.
